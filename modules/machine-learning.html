01

Supervised Learning, Unsupervised Learning, and Reinforcement Learning

What are Supervised Learning, Unsupervised Learning, and Reinforcement Learning?

15 min Notes

In supervised learning, the model is trained using a set of labeled data, meaning that the inputs and corresponding outputs are already determined. The model learns the mapping from input to output and applies it to new data.  Example: Regression models used to predict house prices, or classifying images of dogs and cats based on labeled data. In unsupervised learning, there are no pre-labeled outputs. The task of the model is to find patterns or structures in the data. This is commonly used for data clustering or dimensionality reduction. Example: Clustering algorithms like K-means are used for grouping data. Reinforcement learning models learn how to make decisions through interaction with the environment, with the goal of maximizing cumulative rewards through rewards and penalties. Example: Autonomous vehicles learning in traffic environments, or AlphaGo's application in playing Go.

Key terms: Agent, Environment, Rule‑based systems, Data‑driven systems

02

Data Pipeline

Dataset → Preprocessing → Training → Prediction

15 min Notes

Data is the foundation of machine learning. Raw data usually comes from multiple sources, which can include text, images, or time series data. Preprocessing involves cleaning the data (removing missing values and outliers), as well as normalizing or standardizing the data to make it suitable for training machine learning models. During the training phase, the model learns the relationship between the input data's features and labels, updating its weights and parameters to optimize predictions. Once the model is trained, it can be used to make predictions on new data. The model is applied to a test dataset, generating the corresponding outputs.

Key terms: Agent, Environment, Rule‑based systems, Data‑driven systems

04

Overfitting and Underfitting

What are the Visual Explanation of Overfitting and Underfitting?

15 min Notes

Overfitting occurs when a model performs well on training data but poorly on new data. The model becomes too complex, memorizing the noise in the training data rather than generalizing the patterns. Visual Example: Show the difference between training data points and the model's predictions. Overfitted models have curves that are too bent and cannot adapt to new data. Underfitting happens when a model is too simple and cannot capture the patterns in the data, leading to poor performance on both training and test datasets. Visual Example: Show a simple linear model that cannot fit the data well, leading to poor predictions.
